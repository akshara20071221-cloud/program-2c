### AIM:
To write a C program to add two numbers using functions .
---

### ALGORITHM:
1.Get two numbers as inputs from the user.
2.Write a function to add those two numbers and call the function using the given numbers as arguments.  

---

### PROGRAM:

```c
#include <stdio.h>
int nis(int ,int);
int nis(int n1 ,int n2)
{
    printf("The Result of Addition is:%d",n1+n2);
    return 0;
}
int main()
{
    int num1,num2;
    scanf("%d %d",&num1,&num2);
    nis(num1,num2);
}
 
```
